Property: (default value)

- Type: (None)
   Package type.

- Width: (4 Cells)
   WIdth in grid cells.

- Height: (8 Cells)
   Height in grid cells.

- Name: (Package)
   Name shown at the center of the component.
   Leave empty to show nothing.
   use "Package" to use package file name.

- Package File: ()
   Path to package file.

- Background: ()
   File for background image.

- Logic Symbol (yes)
   If yes, use A logic symbol representation.
   If no, use a "Chip" representation.
