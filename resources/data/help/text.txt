Configurable Text Box.

Property: (default value)

Main:
- Margin: (5 Pixels)
   Space betwen text and border.

- Border: (1 Pixels)
   Width of border line.

- Background Color: (ffffdc)
   Click on the color box to choose a new background color.

- Opacity: (1)
   Value from 0 to 1:
   0 for transparent to 1 for 100% opaque.

Text:
- Font: (Helvetica [Cronyx])
   Font family

- Font Color: (000080)
   Click on the color box to choose a new font color.

- Font Size (10 Pixels)
   Font size in pixels.

- Fixed Width: (yes)
   Charcters have constant width.

- Text Box:
   Edit the text here.
